# 首页（Home）

首页主要包括大标题组件和热门/推荐组件。

- 视图类：DJESiteHomeView

## 模板API

| 特性 | 模板              | 变量名称       | 类型                        | 说明               |
| ---- | ----------------- | -------------- | --------------------------- | ------------------ |
| 首页 | {theme}/home.html | jumbotron      | `starter.widgets.Jumbotron` | 大标题             |
|      |                   | top_chart_list | `List[DJEChartInfo]`        | 热门推荐的图表信息 |
